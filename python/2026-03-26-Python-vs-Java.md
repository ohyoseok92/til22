### Context
* 파이썬과 자바는 인기 있는 프로그래밍 언어입니다.
* 두 언어 모두 객체 지향 프로그래밍을 지원합니다.
### Core
```python
# 파이썬 예시
print("Hello, World!")
```
```java
// 자바 예시
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
### Insight
* 파이썬은 동적 타입语言이며, 자바는 정적 타입 언어입니다.
* 파이썬은 데이터 분석과 머신 러닝에 많이 사용되며, 자바는 안드로이드 앱 개발과 엔터프라이즈 소프트웨어 개발에 많이 사용됩니다.
