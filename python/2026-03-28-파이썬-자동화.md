### Context
* 파이썬自動化은 다양한 작업을 효율적으로 처리할 수 있는 강력한 도구입니다.
* 자동화의 필요성과 파이썬의 역할을 이해하는 것이 중요합니다.

### Core
```python
import os
import time

def auto_rename_files(path):
    # 파일 이름을 자동으로 바꾸는 함수
    for filename in os.listdir(path):
        # 파일 이름을 수정합니다.
        new_filename = filename.replace(" ", "-")
        os.rename(os.path.join(path, filename), os.path.join(path, new_filename))
        print(f"{filename} -> {new_filename}")

# 사용 예시
auto_rename_files("./files")
```

### Insight
* 파이썬의 `os`와 `time` 모듈을 사용하여 자동화 작업을 수행할 수 있습니다.
* 파일 이름을 자동으로 바꾸는 예시 코드를 통해 파이썬의 활용도를 이해할 수 있습니다.
