### Context
*   파이썬 리스트 컴프리헨션은 반복문과 조건문을 사용하여 리스트를 생성하는 방법입니다.
*   리스트 컴프리헨션은 기존 리스트나 다른 iterable을 사용하여 새로운 리스트를 생성할 수 있습니다.

### Core
```python
numbers = [1, 2, 3, 4, 5]
squared_numbers = [num ** 2 for num in numbers]
print(squared_numbers)  # [1, 4, 9, 16, 25]
```

### Insight
*   리스트 컴프리헨션은 기존 리스트나 다른 iterable을 사용하여 새로운 리스트를 생성할 수 있습니다.
*   리스트 컴프리헨션은 반복문과 조건문을 사용하여 리스트를 생성하는 방법입니다.
*   리스트 컴프리헨션은 코드를 간결하게 만들어 주고, 가독성을 높여줍니다.
